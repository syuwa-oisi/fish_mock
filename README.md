# Idle Angler — 釣果が経済になる放置釣りゲー（モック）

画面の四隅に常駐させて、釣り人を眺めながら遊ぶサブゲームのモックです。
釣った魚は単なるスコアではなく **「餌・現金・恒久バフ・一時バフ・装備ドロップ源」** として機能する多用途資源で、
振り分けの選択がそのままプレイスタイルと経済を分岐させます。

## 動かし方

`idle_angler_mock.html` をブラウザで開くだけ（ビルド不要・依存なし）。

ローカルサーバー経由なら:

```
python -m http.server 5566
# → http://localhost:5566/idle_angler_mock.html
```

## 遊び方

- **常駐するのは釣り人と水面だけ**のボーダレス表示（ウィンドウ枠・パネルなし）。
  マウスを乗せると **×1**（デモ用倍速 ×1/×4/×20）/ **📌**(四隅⇆中央の配置切替) / **☰**（メニュー）が現れる。
- **☰** から 魚箱・装備・水槽図鑑・市場・設定 をポップアップで開く。
  釣り場・餌・自動餌化・配置・倍速の切替は「設定」ポップアップから。
- 放置で自動的に釣りが進む。**オフライン進行あり**（最大8時間/400匹、localStorageセーブ）。
- 釣れた魚は「魚箱」から4つの使い道に振り分け:
  - **餌** … 餌Tierに変換（C→小魚餌 / R→活き餌 / E,L→特上餌）。高Tier餌ほどレア魚が釣れる**格上げ連鎖**。頂点魚「深淵の主」は特上餌でのみ釣れる。
  - **捌** … 即現金化。たまに装備（個体差ステ付き）を**飲み込んでいる**。
  - **飾** … 水槽に入れて**恒久バフ**（レアリティ×サイズで効果変動）。図鑑コンプでもボーナス。
  - **放** … 放流して**一時バフ**（レアリティで効果と時間が変動）。
- 釣り場は 渓流 → 港・汽水 → 深海船上 をゴールドで解放。天候（晴/雨/霧）と時間帯（朝昼夕夜）で出現が変わる（例: ヌシは雨限定、タチウオは夜限定）。

## RMT/外部マーケットの扱い（本モックの核心コンセプト)

RMT本体の実装はコストが高いため**意図的にオミット**し、代わりに:

- ゲーム内マーケットを `createLocalMarketSim()`（ローカル需給シミュレーション）として**完全に分離**。
- 需要係数が確率的にドリフトし、プレイヤーの売買でも価格が動く＝「釣果の振り分け→需要の偏り→価格変動」のループをローカルで体験できる。
- 将来 Steam Market / 外部RMT基盤に接続する際は、**同じ MarketAPI 形状（quotes / listings / buy / sellXxx）の非同期アダプタに差し替えるだけ**でゲーム本体は無変更、という設計を見せるのが狙い。

取引対象＝飲み込み装備（個体差）・観賞用希少魚・上位餌、シンク＝放流バフ・竿強化・手数料10%、という経済の骨格はモック内で一通り回ります。

## ネイティブ版（idle-angler/）

```
cd idle-angler
npm install
npm run tauri dev     # 開発起動
npm run tauri build   # NSISインストーラ生成 → src-tauri/target/release/bundle/nsis/
```

- ウィンドウは **透過・枠なし・常時最前面・タスクバー非表示**。常駐するのは釣り人と水面だけ。
- 終了は**システムトレイ**の「終了」から。トレイ左クリックで前面表示。
- 📌・設定の配置ボタンでモニタのワークエリア四隅＋中央へスナップ。情景（Canvas）をドラッグして自由移動も可。
- ☰のポップアップ表示中はウィンドウを一時拡張し、閉じると情景サイズへ戻る。
- **✨スキル**: コインで恒久強化を習得（速釣り/幸運の釣り糸/大物狙い/商才/一本釣りの極意=ダブルヒット/魚箱拡張/自動餌化術/置き竿の心得=オフライン上限）。
- **🎣装備画面は2ペイン**: スロット盤面（竿+3部位）→ 詳細パネル（比較・装着・売却）＋インベントリ。
- **🌊大海モード（コーナーフィット）**: 設定から切替。全配置に対応:
  - **下隅(↘↙)**: マージン0で吸着し、水面が画面の下端・側端まで断ち落ち
  - **上隅(↖↗)**: 空＋海の**ジオラマカード**が天面・側端まで断ち落ち（時間帯別の空・流れる雲・太陽/月・星・空に降る雨）
  - **中央/自由(◎)**: 左右両端がフェードする**パノラマ帯** — 海が左右へ無限に続いて見える
  - いずれも霞・遠波・帆船・海鳥つき。右側の隅では左右反転し、海の彼方が画面外側を向く。
- 演出: 釣果・報酬のフローティングテキスト、エピック以上の釣り上げ閃光、キャスト進行バー、レアリティ発光、C/R一括捌き。

## やりこみ要素（td_mock の大型改修を参考に移植）

- **恒久パッシブ（∞）**: 速釣り/幸運/大物/商才/置き竿は**上限なし** — 効果は線形（+1.5〜2%/Lv）、価格は指数（×1.17〜1.2/Lv）。続ければ確実に強くなるが対数的に逓減し、数値爆発しない。
- **装備レアリティ7段階**: コモン〜レジェンドの上に **不滅/星霜/神話**。上位は固定の超低確率（ミシック≈1/20,000）の「自慢級」で、引けたこと自体が誇り。
- **⚒ 鍛冶**: 捌くと素材（🔹鱗=C/R、🔷虹色の鱗=E/L、🦪深海の真珠=ヌシ）。装備詳細から**強化**（+8%/回・高Lvほど失敗しやすい・失敗で素材消失）と**研磨**（性能再抽選）。
- **⚗ 合成**: 同レア9個→確率で1ランク上（70%→…→6%、失敗は1個に圧縮）。弱い個体から自動投入（強化済みは温存）。
- **👑 ヌシ（主）イベント**: 7〜12分ごとに巨影が出現。成功率は サイズ/レア率/熟練 で上昇（20%→最大90%）。勝てば特大個体（換金1.5倍）＋真珠。
- **📈 釣り場熟練度**: 釣るほどその釣り場の速度・レア率+0.5%/Lv（最大Lv60）。
- **バランス**: 中枢定数は `src/balance.ts` に集約。`node scripts/sim.mjs --hours 24` のヘッドレスシムで進行帯域を検証済み（30分×1.5 → 1h×2.0 → 8h×3.0 → 24h×3.3、ヌシ≈収入の2〜3割）。
- セーブは `%APPDATA%\com.pentronic.idleangler\save.json`（Rustコマンド `save_game`/`load_game`）。
- コード構成: `src/types.ts / data.ts / game.ts / market.ts / render.ts / ui.ts / winctl.ts / main.ts`。
  **市場は `MarketAPI`（非同期）経由のみ** — `market.ts` の `createSteamMarketAdapter()` スタブを実装して
  差し替えれば外部経済（RMT/Steam Market）に接続できる構成。

## 既知の制限（デモ）

- ポップアップ展開中はウィンドウ矩形が広がるため、その透明部分の下にあるデスクトップはクリックできない（閉じれば元どおり）。
- Google Fonts 読み込みはオンライン時のみ（オフラインはシステムフォントにフォールバック）。
